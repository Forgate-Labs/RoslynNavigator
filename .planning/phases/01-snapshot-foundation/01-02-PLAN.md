    - `accesses_db`
    - `filters_by_tenant`

    4) Methods calling repository/database-like APIs are tagged with `accesses_db = 1`.
    5) Tenant-filter pattern (identifier/member containing `tenant`) sets `filters_by_tenant = 1`.
    6) Running extractor twice refreshes rows idempotently for same solution (no uncontrolled duplication).
  </behavior>
