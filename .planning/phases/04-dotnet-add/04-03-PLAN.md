    Insert a new `dotnet add` subcommand group after the existing `dotnetScaffoldCommand` block and before `dotnetCommand.AddCommand(dotnetScaffoldCommand)`.

    Follow the exact same pattern as the scaffold subcommands: distinct Argument variable names to avoid scope conflicts.

    **dotnet add using** — 2 positional args: path, namespaceName
    ```
    var addUsingPathArg = new Argument<string>("path", "Path to the C# file");
    var addUsingNamespaceArg = new Argument<string>("namespace", "Namespace to add as using directive");
    var addUsingSubcommand = new Command("using", "Add a using directive (no-op if already present)");
    addUsingSubcommand.AddArgument(addUsingPathArg);
    addUsingSubcommand.AddArgument(addUsingNamespaceArg);
    addUsingSubcommand.SetHandler(async (string path, string ns) => {
        try {
            var result = await DotnetAddCommand.ExecuteUsingAsync(path, ns);
            Console.WriteLine(JsonSerializer.Serialize(result, jsonOptions));
        } catch (Exception ex) {
            OutputError("dotnet_add_using_error", ex.Message);
            Environment.ExitCode = 1;
        }
    }, addUsingPathArg, addUsingNamespaceArg);
    ```

    **dotnet add field** — 5 positional args: path, className, access, type, name
    Builds content `"<access> <type> _<name>;"` and calls `DotnetAddCommand.ExecuteMemberAsync(path, className, "field", content)`
    ```
    var addFieldPathArg = new Argument<string>("path", "Path to the C# file");
    var addFieldClassArg = new Argument<string>("className", "Name of the target class/record/struct");
    var addFieldAccessArg = new Argument<string>("access", "Access modifier (e.g., private, public)");
    var addFieldTypeArg = new Argument<string>("type", "Field type (e.g., int, string, ILogger)");
    var addFieldNameArg = new Argument<string>("name", "Field name (underscore prefix added automatically)");
    var addFieldSubcommand = new Command("field", "Add a field to a class/record/struct");
    // ... AddArgument x5, SetHandler builds content and calls ExecuteMemberAsync(path, className, "field", $"{access} {type} _{name};")
    ```

    **dotnet add property** — 5 positional args: path, className, access, type, name
    Builds content `"<access> <type> <Name> { get; set; }"` (Name is PascalCase — use the name arg as-is since the caller provides the casing)
    Calls `DotnetAddCommand.ExecuteMemberAsync(path, className, "property", content)`
    ```
    var addPropertyPathArg = ...
    var addPropertyClassArg = ...
    var addPropertyAccessArg = ...
    var addPropertyTypeArg = ...
    var addPropertyNameArg = ...
    var addPropertySubcommand = new Command("property", "Add an auto-property to a class/record/struct");
    // content: $"{access} {type} {name} {{ get; set; }}"
    ```

    **dotnet add constructor** — 3 positional args: path, className, content
    The content arg is the full constructor body including signature, e.g., "public MyClass(ILogger logger) { _logger = logger; }"
    Calls `DotnetAddCommand.ExecuteMemberAsync(path, className, "constructor", content)`
    ```
    var addCtorPathArg = new Argument<string>("path", ...);
    var addCtorClassArg = new Argument<string>("className", ...);
    var addCtorContentArg = new Argument<string>("content", "Full constructor source (signature + body)");
    var addCtorSubcommand = new Command("constructor", "Add a constructor to a class/record/struct");
    ```

    **dotnet add method** — 3 positional args: path, className, content
    The content arg is the full method body including signature, e.g., "public void Greet() { Console.WriteLine(\"Hi\"); }"
    Calls `DotnetAddCommand.ExecuteMemberAsync(path, className, "method", content)`
    ```
    var addMethodPathArg = new Argument<string>("path", ...);
    var addMethodClassArg = new Argument<string>("className", ...);
    var addMethodContentArg = new Argument<string>("content", "Full method source (signature + body)");
    var addMethodSubcommand = new Command("method", "Add a method to a class/record/struct");
    ```

    After all 5 subcommands are defined:
    ```
    var dotnetAddCommand = new Command("add", "Add members to existing C# types");
    dotnetAddCommand.AddCommand(addUsingSubcommand);
    dotnetAddCommand.AddCommand(addFieldSubcommand);
    dotnetAddCommand.AddCommand(addPropertySubcommand);
    dotnetAddCommand.AddCommand(addCtorSubcommand);
    dotnetAddCommand.AddCommand(addMethodSubcommand);
    dotnetCommand.AddCommand(dotnetAddCommand);
    ```

    Place this block between the scaffold subcommand definitions and `dotnetCommand.AddCommand(dotnetScaffoldCommand)`. The dotnetCommand.AddCommand calls go together at the end of the dotnet group block.
