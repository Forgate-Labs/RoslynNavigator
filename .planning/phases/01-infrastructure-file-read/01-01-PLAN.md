Create `src/RoslynNavigator/Services/BackupService.cs` with namespace `RoslynNavigator.Services`:

`BackupService` is a non-static class (instantiable, for testability). Constructor takes `string workingDirectory`.

Public method:
```csharp
public async Task<string> CreateBackupAsync(IEnumerable<string> filePaths)
```

Implementation:
1. Generate timestamp string: `DateTime.UtcNow.ToString("yyyyMMdd-HHmmss")`
2. Compute `backupDir = Path.Combine(workingDirectory, ".roslyn-nav-backup", timestamp)`
3. `Directory.CreateDirectory(backupDir)`
4. For each `filePath` in `filePaths`:
   - Resolve to absolute path: if not rooted, `Path.GetFullPath(Path.Combine(workingDirectory, filePath))`
   - Skip files that do not exist (new files being created have nothing to back up)
   - Compute relative path from `workingDirectory` using `Path.GetRelativePath(workingDirectory, absolutePath)`
   - Compute destination path: `Path.Combine(backupDir, relPath)` — ensure parent dir exists with `Directory.CreateDirectory(Path.GetDirectoryName(destPath)!)`
   - `File.Copy(absolutePath, destPath, overwrite: true)`
5. Return `backupDir` (the path of the created backup directory)

Also add a static factory: `public static BackupService CreateDefault() => new BackupService(Directory.GetCurrentDirectory());`

No usings needed beyond `System.IO` (implicit via ImplicitUsings in .NET 10).
